Classes

.h file 
@interface ClassName : Superclass {
  // Instance Variables
}
  // Public Methods
@end
.m file 
@implementation ClassName
  // Method Implementations
@end

Instance Variable Directives

  • @protected - default for ivars defined in @interface
  • @private - default for ivars defined in @implementation
  • @public
  • @package

Properties

Properties are essentially accessors for instance variables.

Categories

Categories are used to extend existing classes, especially when you don't have access to the existing source code.

.h file (convention: ClassName+CategoryName.h) 
#import "ClassName.h"

@interface ClassName (CategoryName)
  // Methods Only (No ivars)
@end
.m file 
@implementation ClassName (CategoryName)
  // Method Implementations
@end

Class Extensions

A category defined without a name is a "class extension." Class extensions allow you to define additional instance variables.
